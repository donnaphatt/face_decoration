# Face filter

In this project, I practice to use the mediapipe library for face and face landmark detection.

Moreover, I attempt to implement kind of AR project too. :D

**Requirement**
- Python 3.7
- opencv, matplotlib, mediapipe, Numpy

**Run**
- `python main.py`
